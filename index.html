Kitty Ice Cream Cafe View Here
Tech Stack HTML5 · CSS3 · Vanilla JavaScript · localStorage API · Netlify
Retrospective Extended a five-page static cat cafe site by designing and building a fully client-side community recommendation board from scratch. The core challenge was delivering post, delete, and like interactions with zero backend — everything lives in localStorage, with a version key that flushes stale data whenever the menu changes. Locked the submission form to the real menu via a dropdown that auto-categorizes entries on select, added manual HTML escaping throughout all user-generated rendering to prevent XSS, and pre-seeded the feed so the page is never empty on first visit. Also updated the global nav and footer across all six pages and shipped a version-controlled data reset pattern so future menu edits don’t leave orphaned posts.
